# DOTCOM2022

## Introduction

DOTCOM2022是一个DOTCOM校内网站计划，为 Hainan University DOTCOM 创建属于自己的官网。

## Work and Function

- 记录每次培养计划的授课内容
- 挂载所有资源
- 锻炼成员Web开发能力(Main Purpose)
- ~~我们终于有自己的官网了~~

## Contribution

欢迎共同维护，在此之前，需要具备：

- 一个Github账号
- 一定的HTML、CSS语法基础，一定的Web前端（视图）框架基础
- 热枕的心
